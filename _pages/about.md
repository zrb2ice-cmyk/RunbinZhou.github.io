Search Infrastructure Engineer specializing in High-Scale Recall Systems
======
Architect of low-latency, high-throughput retrieval pipelines for large-scale search engines. Core competencies:
* Recall Architecture: Designing distributed inverted index and ANN systems (HNSW, IVF, etc.)
* Performance Engineering: Cache-optimized data structures, lock-free concurrency, and modern C++ optimization
* Full-Pipeline Optimization: Offline index building, online serving tradeoffs, and cost-efficient resource utilization
* Industrial Search Platforms: Building evolvable core retrieval layers supporting billion-scale documents

Specialized in balancing recall quality with strict latency/SLO requirements in production environments.
